HTML

  • HTML is the structure for a webpage
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section element is a semantic element that defines a section in a document that contains thematic content
  • The ul element is used to create an unordered list of things
  • The li element is used to create a list item inside a listed element

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Using the "*" selector declares that ALL of the elements on the page will have the style properties
  • display: block;: Value which assigns each image to start on a new line

Git

  • git init: Initialize an existing directory as a Git repository
  • git status: checks what branch we are currently on
  • checkout -b "branch name": Creates a new branch and switches to it
  • git add -A: Adds/Stages all changes to the code to the current working branch
  • git checkout: Switch from one branch to another
  • git commit -a: Commits files you've add with "git add"
  • git commit -m "commit message": Commit changes to head (but not to the remote repository)
  • git push origin master: Sends changes to the master branch of your remote repository
  • List of basic Git commands: here

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Strict Equality "===": Operator that is used to test to see if two values are equal and of the same data type. It returns a true/false (Boolean) result
  • Not, Does-Not-Equal "!", "!==": Tests to see if values are NOT equal. Returns a "false" if value is not equal
